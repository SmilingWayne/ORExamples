# ORExamples

This repo contains some of my codes during learning of OR.

- [Exam schedule (demo)](./Notebooks/ExamSchedule/): A very simple yet interesting exam schedule based on integer program.
- [Circle Pack](./Notebooks/CirclePack/): To Pack circles in a bigger circle, based on [circlify](https://github.com/elmotec/circlify), support multi-level packer. Also refer to [d3.js](https://d3js.org/d3-hierarchy/pack).
- [Rectangle Pack](./Notebooks/RectPack/): To Pack rectangles in a bigger rectangle! Set cover and Guillotine cut supported.
- [Minimum Circie Cover](./Notebooks/MinCircleCover/):  To cover vertices in a bigger circle. Support larger dimension.
